# CalculatorAndroidApp
Calculator App for Android Phones implemented in Kotlin with support for basic operations like addition, subtraction, division, multiplication and remainder.

## Tools
* Android Studio IDE
* Rhino (for evaluating expressions)
* Kotlin
* Android SDK 21 - 32

## Instructions for Running The Project
* Open the Project in Android Studio
* Build and Run on Emulator or a Compatible Mobile Device
